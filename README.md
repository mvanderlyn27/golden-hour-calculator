# golden-hour-calculator
https://photo-spot.vanderlyn.space
## purpose
web app to calculate golden hour at any time, anywhere on the planet
## stack
* kubernetes: k3s
* frontend: 
  * React
  * Typescript
  * fluent ui
* backend:
  * Go
  * GIN
